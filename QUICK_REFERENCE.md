- **效果**: 格式保留率 ↑ 80%+

- **改进**: 明确角色、要求引用、处理不足
- **效果**: 响应准确度 ↑ 50%+

## 🚀 快速启动

```bash
# 1. 安装依赖
cd backend
pip install -r requirements.txt

# 2. 配置 .env (硅基流动 API 密钥)
EMBEDDING_API_KEY=your_key_here

# 3. 启动后端
python -m uvicorn main:app --host 0.0.0.0 --port 8000

# 4. 启动前端
cd frontend
npm run dev

# 5. 访问应用
# http://localhost:3000
```
