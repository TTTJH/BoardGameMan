## 💡 使用建议

### ✅ 推荐做法

1. **提出具体问题**
   - 好: "How do you gain experience points?"
   - 差: "Tell me about the game"

2. **上传完整 PDF**
   - 确保 PDF 格式正确
   - 支持中文、德文、英文混合

3. **等待首次嵌入**
   - 首次查询会生成嵌入（较慢）
   - 后续查询会使用缓存（快速）

4. **查看源文档**
   - 始终检查引用的源文档
   - 验证答案的准确性

### ❌ 避免做法

1. 提出模糊问题
2. 上传不完整的 PDF
3. 频繁刷新页面
4. 忽视错误信息

## 🔧 故障排除

### 问题 1: 嵌入 API 错误

**症状**: "API key invalid" 或 "Connection refused"

**解决**:
1. 检查 `.env` 中的 API 密钥
2. 确保网络连接正常
3. 验证 API 端点是否可访问

### 问题 2: PDF 提取失败

**症状**: "Failed to extract text" 或 "Invalid PDF"

**解决**:
1. 确保 PDF 格式正确
2. 尝试重新上传
3. 检查文件大小 (< 50MB)

### 问题 3: 搜索结果不相关

**症状**: "Results don't match the question"

**解决**:
1. 尝试更具体的问题描述
2. 检查 PDF 是否包含相关内容
3. 查看源文档引用

### 问题 4: 响应时间过长

**症状**: "Request timeout" 或 "Waiting for response"

**解决**:
1. 这是正常的，首次查询需要生成嵌入
2. 后续查询会更快
3. 检查网络连接
